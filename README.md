# vishakbaddur.github.io

Personal portfolio website.

Built as a single HTML file: no frameworks, no build step, no dependencies.

**Live:** https://vishakbaddur.github.io

## What's in it

- Terminal bar with rotating status messages
- Fixed control panel with live activity log and stats
- Problem-Struggle-Resolution narrative for every project and role
- All 8 marimo merged PRs documented
- Operating System section — engineering philosophy meets calisthenics
- Fully responsive, works on mobile

## Stack

Plain HTML, CSS, JavaScript. IntersectionObserver for scroll animations. No libraries.

## Update

Edit `index.html`, commit to main. GitHub Pages redeploys in ~30 seconds.
