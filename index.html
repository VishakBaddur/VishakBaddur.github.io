// open_source

The bugs I couldn't leave alone.

marimo is a reactive Python notebook used by thousands of AI/ML researchers. I found bugs. I fixed them. Eight times. view all 8 PRs

  • Fixed disabled cells permanently displaying ancestor errors after recovery MERGED

    3-layer root cause: wrong backend code path, stale snapshot timing, transitionCell() not clearing the errored flag. Fix moves recovery into _run_cells() with a pre-run snapshot, broadcasts broadcast_empty_output, patches the frontend state machine. 24 commits. 4 maintainers.

    PR #8784
  • Fixed RuffFormatter applying wrong blank-line rules to cell code MERGED

    ruff treats stdin as file scope (E302) but on disk cells are wrapped in def _(): (E301). Wrapped cells before formatting, unwrapped via AST parsing, used skip_wrap set for comment-only cells. Pushed back on maintainer's suggested fix with an empirical counterexample and won.

    PR #9889
  • Built syntax highlighting for Python code blocks in LSP hover docs MERGED

    Traced the full render path from LSP response through formatContents() to DOM. Injected a custom markdownRenderer using @lezer/python and @lezer/highlight with light-dark() theming. Zero new dependencies. 5019 tests pass.

    PR #9824
  • Fixed autoreload performance regression: 12ms/cell to 0.17ms (62x) MERGED

    _is_user_module() called os.path.realpath on every module on every check(). Added @functools.cache-decorated _normalized_path() memoizing by __file__ string. 62x speedup on a 20-cell notebook.

    PR #10086
  • Fixed mo.status.spinner silently lost inside vstack/hstack MERGED

    Eager HTML serialization in _FlexContainerHtml baked frozen strings at construction time. Redesigned to hold live child references and rebuild on every flush.

    PR #8626
  • Fixed --sandbox mode failure for relative path sources MERGED

    uv export --script returns script-relative paths that broke when resolved against CWD. Converted to absolute using the script's parent directory as base in _uv_export_script_requirements_txt().

    PR #9052
  • Resolved pyright type error in mo.ui.table(chart.value) MERGED

    data parameter only declared IntoDataFrame, missing the other ChartDataType members. Expanded TableData alias. Ruled out lazy auto-detection after it broke 8 parametrized tests across Polars and Ibis fixtures.

    PR #9674
  • Fixed hover tooltip duplication for all LSP providers, not just pylsp MERGED

    hintTooltip() only suppressed marimo's built-in hover when pylsp was active. ty, pyrefly, and basedpyright still fired duplicate tooltips. Extended the guard to cover all 4 providers.

    PR #9741
